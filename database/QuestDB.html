# 简介

QuestDB 是一个面向关系列的数据库，专为时间序列和事件数据而设计.

特点: 快/兼容PostgreSQL 的 wire 协议，也兼容 InfluxDB Line 协议

用途: 基于ts的快照存储,高性能查询检索.
